# 酷鸽商城后端服务

1. H5端：http://124.71.130.245/h5
2. 管理端：http://124.71.130.245/admin

kuge-mall是一套完整的在线购物系统，包括C端和管理端，客户可在C端体验商品浏览、下单支付、申请售后、参与秒杀、使用优惠券等完整的购物流程，管理员可在后台创建商品、定制营销规则、管理订单，管理店铺、创建活动，对商品、订单、用户进行管理；技术栈为SpringBoot、SpringCloud、SpringGateway、MySQL、Redis、RabbitMQ、JWT、Nacos、OpenFeign、JSR303

## 项目模块
``` lua
mall
├── mall-admin        -- 后台管理服务
├── mall-auth         -- 权限认证服务
├── mall-cart         -- 购物车服务
├── mall-common       -- 公用模块
├── mall-gateway      -- 网关服务
├── mall-member       -- C端用户服务
└── mall-order        -- 订单服务
└── mall-product      -- 商品服务
└── mall-sale         -- 营销服务
└── mall-thirdaprt    -- 第三方集成服务
```

## maven配置
1. 下载 maven
2. 配置阿里镜像源
   ```shell
   <mirror>
      <id>alimaven</id>
      <name>aliyun maven</name>
      <url>http://maven.aliyun.com/nexus/content/groups/public/</url>
      <mirrorOf>central</mirrorOf>        
   </mirror>
   ```
3. 设置自定义本地仓库地址
   ```shell
   <localRepository>path/to/repository</localRepository>
   ```
